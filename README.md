# 🏥 Doctor AI – Intelligent Healthcare Assistant

An AI-powered healthcare assistant that enables users to interact with a virtual doctor using **chat, voice (STT), and audio (TTS)**.  
Built using modern AI stack including **LLMs, RAG, LangChain, LangGraph, and vector database (Pinecone)**.

---

## 🚀 Live
https://healthcaresystem2-1.onrender.com
Example: https://doctor-ai.onrender.com

---

## 📌 Features

- 🧠 AI Doctor Chat (LLM-powered responses)
- 🎤 Speech-to-Text (Voice input)
- 🔊 Text-to-Speech (Voice output)
- 📚 RAG-based medical knowledge retrieval
- 🔍 Vector search for accurate answers
- 🧩 Context-aware conversation memory
- 📱 Mobile-ready UI (Capacitor compatible)
- 🌐 Full-stack architecture
